# Abdirashid Omar

Personal website for Abdirashid Omar, built with Jekyll and the AcademicPages / Minimal Mistakes style.

The site includes publications, data stories, blog posts, and downloadable CV versions for Vision AI and Data Analyst roles.

## Local Development

```bash
bundle install
bundle exec jekyll serve --host 127.0.0.1 --port 4000
```

The generated `_site/` folder, local story repositories, bundled gems, and CV source folders are intentionally ignored so they are not pushed to GitHub Pages.
